Client TLS context to use for any secure network connections made while sourcing credentials (for example, a profile that uses assume-\/role will need to query STS).

If a TLS context is needed, and you did not pass one in, it will be created automatically. However, you are encouraged to pass in a shared one since these are expensive objects. If using BYO\+\_\+\+CRYPTO, you must provide the TLS context since it cannot be created automatically.
