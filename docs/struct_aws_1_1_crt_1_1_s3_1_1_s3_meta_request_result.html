Detailed Description

Result delivered to FinishCallback when a meta request terminates. Mirrors aws_s3_meta_request_result. errorResponseHeaders is a deep copy you may retain freely. errorResponseBody is a borrowed view into CRT-owned memory, valid only for the duration of the FinishCallback invocation; copy out its contents if you need to retain them.
